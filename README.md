
# 🧍 Procedural Tentacle Animation (Blender)

This project demonstrates a **procedural animation setup** in Blender, where flexible, spine-like appendages dynamically move around a humanoid rig. The animation showcases organic motion using curves, constraints, or rigging techniques.

## 🚀 Features

* Procedural tentacle/appendage animation
* Smooth, organic motion using rigging techniques
* Multiple independent animated limbs
* Clean humanoid base model for demonstration
* Easily customizable movement and controls

## 🛠️ Tools Used

* Blender (Animation & Rigging)
* Basic rigging (Armature / Curve modifiers / Constraints)

## ⚙️ How It Works

* The humanoid model acts as the base.
* Tentacles are controlled using:

  * Armatures OR
  * Curve modifiers with control points
* Motion is achieved through keyframes and/or procedural drivers.
* The setup allows smooth bending and dynamic movement.

## ▶️ How to Use

1. Open the `.blend` file in Blender
2. Play the timeline to view animation
3. Select control rigs to modify movement
4. Adjust keyframes or constraints for custom animation

## 🎯 Use Cases

* Game animation prototypes
* Creature design experiments
* Procedural animation learning
* VFX and motion studies
